API DO SISTEMA SÉSAMO

Passo a Passo para rodar o código.
1 - Após clonar o repositório, ou mesmo baixá-lo, dentro da pasta da api digite o seguinte comando:
npm install

2. Depois que o comando acima baixar todas as dependencias necessárias, a api é iniciada com o comando:
nmp start

Importante lembrar que o Postgres já deve estar inicializado, e um base de dados chamada sesamo já ter sido criada.

A documentação da API está disponível em:
https://documenter.getpostman.com/view/9423486/SW7gU4xf
